# OnlyOffice Web Local

[English](readme.md) | [中文](readme.zh.md)

基于 OnlyOffice 的本地网页文档编辑器，允许您直接在浏览器中编辑文档，无需服务器端处理。

## 功能特点

- 在浏览器中本地编辑文档
- 支持多种文档格式（DOCX、XLSX、PPTX 等）
- 实时文档预览
- 无需服务器端处理
- 注重隐私：所有处理都在浏览器中进行

## 部署

当更改推送到主分支时，项目会自动部署到 GitHub Pages。
